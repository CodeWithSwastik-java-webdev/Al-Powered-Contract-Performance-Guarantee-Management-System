# Design System

## Project

AI-Powered Contract Performance Guarantee (CPG) Management System

---

# 1. Design Philosophy

The application should feel like an enterprise software product used by POWERGRID.

The interface must be:

- Modern
- Minimal
- Professional
- Elegant
- Clean
- Responsive
- Accessible

The application should never resemble a generic admin template or a college project.

Design inspiration:

- Microsoft Azure Portal
- Atlassian
- SAP Fiori
- Notion
- Linear

---

# 2. Theme

Overall Theme

Soft White + Soft Green

The interface should create trust, clarity and professionalism.

Avoid unnecessary gradients.

Avoid flashy colors.

Use subtle animations.

# 28. Design Rules

Always use reusable components.

Maintain consistent spacing.

Never hardcode colors.

Use theme variables.

Never create inconsistent layouts.

Avoid clutter.

Whitespace is important.

Every page should feel balanced.

---

# 29. UX Principles

Reduce user clicks.

Keep important actions visible.

Provide immediate feedback.

Confirm destructive actions.

Use progressive disclosure.

Never overwhelm users.

---

# 30. Golden Rule

Every screen should look polished enough that it could be demonstrated to POWERGRID management without appearing unfinished.

Consistency is more important than adding visual effects.
